Доступные профессии

На проекте RP-IR каждый может найти себе дело по душе. Управляй поездами, руководи движением или просто путешествуй — выбирай свою роль!

Кем можно стать?

Машинист

Главный человек в кабине. Ведёшь состав по маршруту, следишь за приборами, соблюдаешь график. Ответственность и уважение гарантированы.

Помощник машиниста

Правая рука машиниста. Помогаешь вести состав, следишь за путём, учишься у старших товарищей. Лучший старт для новичка.

Диспетчер

Мозг движения. Координируешь поезда, даёшь команды, отвечаешь за безопасность на участке. Только для хладнокровных стратегов.

Дежурный по станции

Встречаешь поезда, работаешь с пассажирами, следишь за порядком на вокзале. Лицо станции и душа компании.

Дежурный по переезду

Обеспечиваешь безопасность на переездах. Закрываешь шлагбаумы, пропускаешь составы, следишь за дорогой. Спокойная и важная работа.

Пассажир

Просто путешествуешь, наслаждаешься видами, общаешься с другими игроками. Никакой ответственности — только удовольствие.
