ChatTFP для Windows

Десктопное приложение для Windows с полным доступом ко всем моделям OpenRouter, удобной системой чатов и поддержкой Markdown. Работает быстрее и не требует браузера.

Windows

Версия 1.0.0 (64-bit)

Мгновенный запуск

Приложение загружается за секунды, не тратит ресурсы на браузерные вкладки. Работает даже при закрытом браузере.

Неограниченные чаты

Создавайте сколько угодно диалогов, группируйте их по темам, ищите по истории. Всё хранится локально.

Продвинутый редактор кода

Подсветка синтаксиса, автоформатирование, вставка сниппетов. Идеально для программистов.

Офлайн-режим

Доступ к локальным моделям (через Ollama) и возможность работы без интернета.

Системные требования (Windows)

Windows
  • Windows 10/11 (64-bit)
  • 4 ГБ RAM (рекомендуется 8 ГБ)
  • 500 МБ свободного места
  • Поддержка SSE4.2 (любой процессор после 2010 г.)
